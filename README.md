## Over dit project
Dit project bevat een kleine chatassistent die in de browser draait. Het is bedoeld als leer-/demo-project en bestaat uit standaard webtechnieken.

## Features
- Chat UI in de browser
- Simpele input/response flow
- Responsive styling (afhankelijk van de huidige CSS)

## Technologie
- HTML
- CSS
- JavaScript

## Installatie
1. Clone de repository:
   ```bash
   git clone https://github.com/boudewijn010/chatassistend-Boiten.git
   ```
2. Open het project:
   - Open `index.html` in je browser
   - Of gebruik een lokale webserver (bijv. VS Code Live Server)

## Gebruik
- Typ een bericht in het invoerveld
- Druk op Enter of klik op verzenden (als aanwezig)

## Projectstructuur
> Bestandsnamen kunnen verschillen; dit is een indicatie.

- `index.html` – hoofdpagina
- `css/` of `styles/` – styling
- `js/` of `scripts/` – logica
- `assets/` – afbeeldingen/icoontjes (indien aanwezig)
